# End-to-end-Data-Pipeline-Airflow-Docker
Data engineering project that builds an automated data pipeline. It uses Python and SQL for data processing, Airflow for scheduling, and Docker for setup. It also includes CI/CD to ensure the code works correctly and updates automatically.

## Roadmap
1. Data Extraction
2. Data Loading & Transformation
3. Data Quality Testing
4. Functional & E2E Testing
5. CI-CD
